# Streama Support Portal
## Description

Support portal for Streama service

## Installation

```bash
$ npm install
```

## Running the app

```bash
# build project
$ npm run build

# start app w/hot reload
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```
